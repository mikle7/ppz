This server is running with PPZ_SERVER_AUTH_MODE=none (the default). The admin web UI is open to anyone who can reach it — intended for trusted-network deployments. Firewall accordingly.

To require username & password login:

  1. Create user(s) on the Users page (Cycle F adds this).
  2. Set PPZ_SERVER_AUTH_MODE=password in your server environment.
  3. Restart ppz-server.
