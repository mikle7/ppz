Get started

You don't have any pipes yet — here's how to create your first one.

  1. Install the ppz CLI: 
    curl -fsSL https://raw.githubusercontent.com/pipescloud/ppz/main/install.sh | bash
  2. Log your daemon into this server (grab an API key from your account's Keys tab): 
    ppz daemon login <server-url> -apikey <your-key>
  3. Claim an actor identity: 
    ppz source create <your-handle>
  4. Create your first pipe and send a message: 
    ppz pipe create lobby
ppz send lobby "hello"

This panel disappears once your first pipe lands.
